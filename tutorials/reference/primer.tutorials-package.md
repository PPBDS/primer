# primer.tutorials: Interactive Tutorials for Preceptor's Primer for Bayesian Data Science

This package provides interactive tutorials that accompany "Preceptor's
Primer for Bayesian Data Science: Using the Cardinal Virtues for
Inference". The tutorials cover the fundamental concepts of data science
through the lens of Bayesian thinking, including data visualization,
wrangling, modeling, and statistical inference using modern R tools.

## Getting Started

To see available tutorials, run:
`learnr::available_tutorials("primer.tutorials")`

To start a tutorial, run:
`learnr::run_tutorial("01-probability", package = "primer.tutorials")`

## About the Primer

These tutorials are designed to complement "Preceptor's Primer for
Bayesian Data Science: Using the Cardinal Virtues for Inference", which
teaches data science through the framework of the four cardinal virtues:
Wisdom, Justice, Courage, and Temperance. The textbook is available at
<https://ppbds.github.io/primer/>.
