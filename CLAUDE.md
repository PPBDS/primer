# CLAUDE.md

The working reference for authoring the *Preceptor's Primer* book chapters,
tutorials, and class exercises now lives in the shared **ai-rules** repo:

- **Primer guide:** https://github.com/PPBDS/ai-rules/blob/main/claude-md/primer/CLAUDE.md

It moved there because the guidance is shared. The Primer guide builds on a
**base tutorial guide** — the default contract for authoring any "normal" data
science tutorial in this ecosystem — and other repos will reuse the same
material (for example, a future `classroom-exercises` repo whose exercises have
much in common with Primer tutorials and chapters):

- **Base tutorial guide:** https://github.com/PPBDS/ai-rules/blob/main/claude-md/tutorials/CLAUDE.md

**Before working on a Primer chapter, tutorial, or class exercise, read the
Primer guide first** (which itself reads in and accepts the base tutorial
guide). Keeping the canonical copy in ai-rules means a change made once
propagates to every repo that points here.
